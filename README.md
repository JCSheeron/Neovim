# Neovim
Neovim configuration related stuff, mostly init.vim
init.vim is usually in ~/.config/nvim, and is analogous to .vimrc

The normal structure is
~~~~
~
 |__/.config
      |
      |__ nvim
	  |    |
	  |    |--init.vim
      |
      |    |__ /ftplugin
      |        |__ <filetype>.vim
	  |
	  |    |--/bundle   (plugins)
~~~~
